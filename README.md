# DSA Practice in Python

Welcome to my Data Structures & Algorithms (DSA) Practice repository! This repo contains my solutions to beginner-level DSA problems in Python, organized by topic. It is designed to track my learning progress and build a portfolio of algorithmic problem-solving skills.

---

## Repository Structure
```text
DSA-Practice/
│
├── Arrays/        # Exercises and problems related to arrays/lists
├── Strings/       # String manipulation and algorithm exercises
├── Recursion/     # Recursive solutions and exercises
├── Sorting/       # Bubble sort, selection sort, etc.
├── Searching/     # Linear search, binary search
├── Stack_Queue/   # Stack and Queue implementations
└── README.md      # Overview of the repository
```

---

## How to Use

- Each folder contains Python files (.py) with individual exercises.

- File naming convention: problem_name.py (e.g., reverse_string.py)

- Each file includes:

    - Problem description

    - Python solution

    - Approach explanation (as comments)

You can clone this repo and run the scripts locally to test the solutions:
```Bash
git clone https://github.com/yourusername/DSA-Practice.git
cd DSA-Practice
python3 Arrays/max_in_list.py
```
